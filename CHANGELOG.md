# Changelog

## 0.1.0

- Initial release
- Support for 8 sports: football, tennis, basketball, rugby league, rugby union, ice hockey, baseball, American football
- CLI commands: `upcoming` and `historic` for scraping matches
- Multiple betting markets support per sport (1x2, over/under, Asian handicap, etc.)
- Local storage in JSON or CSV format
- Remote storage via AWS S3
- Docker compatibility with containerized execution
- Proxy support (HTTP/SOCKS) for enhanced anonymity
- Odds history scraping capability
- Preview mode for faster average odds extraction
- Environment variables configuration support
- AWS Lambda deployment support via Serverless Framework
